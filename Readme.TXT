Proyecto: DPDelivery

Este proyecto esta realizado por:

Alejandro Aguilar Grande
Francisco Javier Borox
David Martinez Bergantiño

Para ejecutar el proyecto se debe ejecutar la clase DeliveryDemo
y se ejecutara los 3 casos posibles (dos de ellos estan comentados, para ejecutarlos descomentarlos)

Consideramos que hemos implementado los siguientes extras:

1. Utilizamos el Iterator en varios casos.
2. Hemos usado tipo Enum.
3. Utilizamos Git en la subida o bajada de archivos grupales del proyecto.
4. Realizamos las pruebas unitarias con JUnit, tanto las obligatorias como las extra.
5. Usamos los mecanismos de excepción (como NullPointerException, IllegalArgumentException
o IllegalStateException).
6. Uso de Interface (PopularEnRedes).
